2D Character Rigging
Prior to my work in 3D rigging, I had been personally interested in and taught myself 2D rigging through creating VTube models- rigged characters that respond to live facial movement.
This model consists of over 100 separate 2D images, each of which have ranges of motion defined by sets of keyframes (an earlier model is disassembled in the middle right image). These keyframes are then evaluated based on facial data from the program VTube studio, and the model then reacts accordingly. As well as that, pendulum-based physics are applied after movement to create secondary motion.
For example, each layer that makes up an eye has keyframes assigned for x, y, and z movement, as well as eyeOpen and eyeSmile. The xyz parameters adjust the position of each layer as the user turns their head, while the others deform its shape while the user blinks or smiles. In order to create a head with 180 degrees of movement, I iterated over several versions to create a design that would function ideally.
While no two models are the same, I've learned how to incorporate designs and technology from previous models into future ones. While the model you see at the bottom looks quite different, I was able to create it substantially faster by using a number of design elements from the model featured above.
