2D Character Rigging
Prior to my work in classes in 3D rigging, I had been personally interested in and taught myself 2D rigging. This particular model is one created for live face-tracking and lip sync.
This model consists of over 100 separate pieces, each of which have ranges of motion defined by sets of keyframes (an earlier model is disassembled in the bottom right image). These keyframes are then evaluated based on facial data from the program VTube studio, and the model then reacts accordingly. As well as that, pendulum-based physics are applied after movement to create secondary motion.
For example, each layer that makes up an eye has keyframes assigned for x, y, and z movement, as well as eyeOpen and eyeSmile. The xyz parameters adjust the position of each layer as the user turns their head, while the others deform its shape while the user blinks or smiles. In order to create a head with 180 degrees of movement, I iterated over several versions to create a design that would function ideally.
